# eth-explorer

Explore the Ethereum blockchain using a rich GraphQL API.

## Usage

[Demo](https://multi-api-jisphshaoi.now.sh/)

```sh
git clone git@github.com:dsys/eth-explorer.git
cd eth-explorer
yarn install
yarn start # listening on localhost:4000
```

## Development

```sh
yarn dev
```
